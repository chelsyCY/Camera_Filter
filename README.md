## Installation
Clone the source locally:

`git clone https://github.com/chelsyCY/Camera_Filter/
cd foco`

Some package that needed to be installed to be run:
`pip install opencv-python
pip install numpy
pip insyall mediapipe`

## Tech Stack
- Python
- OpenCV
- NumPy
- MediaPipe

## Features
- Real-time webcam display
- Emoji/graphic filters applied on detected faces
- Face landmark detection using MediaPipe
- Picture result in PNG
